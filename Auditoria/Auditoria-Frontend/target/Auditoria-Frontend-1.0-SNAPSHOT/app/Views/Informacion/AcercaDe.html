Acerca de
Sistema para la auditoría en imágenes utilizando computación cognitiva de IBM Watson y Google Cloud Vision.

El sistema apoya la gestión de información, brindando soluciones para:
  • Extracción
  • Identificación
  • Clasificación
Su campo de aplicación va dirigido a documentos como:
  • Cédula de ciudadanía
  • Tarjeta de Identidad
  • Formulario de Vinculación
  • Certificado de Ingresos y Retenciones
  • Entre otros
